LineRenderer.GetPositions

Switch to Manual
public int GetPositions (out Vector3[] positions);

Parameters

positions要检索的位置的数组。传递的数组的大小至少应是 positionCount。

Returns

int 实际存储在输出数组中的位置数。

Description

获取线中所有顶点的位置。

在检索所有位置时，此方法优于 GetPosition，因为使用单个命令获取所有位置比单独获取每个位置更高效。

另请参阅：positionCount 属性、GetPosition 函数。
