Unity User Manual (2019.1)

使用 Unity Editor 可创建 2D 和 3D 游戏、应用程序和体验。请从 unity3d.com 下载 Editor。

Unity 用户手册可帮助您了解如何使用 Unity Editor 及其相关服务。您可以从头到尾阅读该手册，或将其用作参考。

如果这是第一次使用 Unity，请查看有关使用 Unity 的介绍性文档，并观看 Unity 教程。
