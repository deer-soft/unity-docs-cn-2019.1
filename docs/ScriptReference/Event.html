Event

class in UnityEngine

Switch to Manual

Description

UnityGUI 事件。

事件与用户输入（按键、鼠标操作）相对应，或者是 UnityGUI 布局或渲染事件。

对于每个事件，在这些脚本中均调用 OnGUI；因此每帧可能多次调用 OnGUI。 Event.current 对应于 OnGUI 调用内的“当前”事件。

另请参阅：GUI 脚本指南、EventType。
