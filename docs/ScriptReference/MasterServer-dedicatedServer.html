Removed

MasterServer.dedicatedServer

Switch to Manual
Obsolete public static bool dedicatedServer ;

Description

将此计算机报告为专用服务器。

如果作为服务器运行，则连接计数定义玩家计数，在主服务器上注册时会报告 此计数。默认情况下，主服务器假设此实例不是专用服务器，因而玩家 计数每次递增一（以考虑在服务器上运行的“客户端”）。如果不需要这样，此 变量可以设置为 false，然后在主机数据中仅将连接计数报告为玩家计数。
