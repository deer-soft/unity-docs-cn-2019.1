混响区

__混响区__获取音频剪辑并根据音频监听器在混响区内的位置使音频剪辑失真。从没有环境效果的点逐渐到有环境效果的点（例如进入洞穴时），便可使用混响区。

属性

属性： 功能：
Min Distance 表示辅助图标中内圆的半径，决定了逐渐出现混响效果的区域和完整的混响区。
Max Distance 表示辅助图标中外圆的半径，决定了没有效果的区域和开始逐渐应用混响的区域。
Reverb Preset 决定了混响区将使用的混响效果。

提示
