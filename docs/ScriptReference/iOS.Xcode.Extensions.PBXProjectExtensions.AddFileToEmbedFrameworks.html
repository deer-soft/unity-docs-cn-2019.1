PBXProjectExtensions.AddFileToEmbedFrameworks

Switch to Manual
public static void AddFileToEmbedFrameworks (iOS.Xcode.PBXProject proj, string targetGuid, string fileGuid);

Parameters

proj作为此参数传递的项目。
targetGuid[[TargetGuidByName()]] 返回的目标的 GUID。
fileGuidAddFile 或 AddFolderReference 返回的文件 GUID。

Description

为给定原生目标的嵌入式框架部分配置文件。

此外，此函数还会在内部调用 proj.AddFileToBuild(targetGuid, fileGuid)，以确保框架会添加到关联框架列表中。

如果目标已将给定文件配置为嵌入式框架，则此函数将不起作用。

在包含多个原生目标的项目中，可以将单个文件或文件夹引用配置为在所有目标或部分目标中构建，或者不在任何目标中构建。根据文件扩展名，文件或文件夹引用会添加到适当的构建部分。
