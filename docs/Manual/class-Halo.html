光环 (Halo)

__光环__是光源周围的亮区，用于给人留下空气中小尘粒的印象。

属性

属性： 功能：
Color 光环的颜色。
Size 光环的大小。

详细信息

您可以将 Halo 组件添加到光源 (Light) 对象，然后设置其大小和颜色属性以提供所需的发光效果。此外，也可将 Light 组件设置为在没有单独 Halo 组件情况下显示光环，为此需启用其 Draw Halo 属性。

提示

  • 要在 Scene 视图中查看光环，请在 Scene 视图__工具栏中选中 Fx__ 按钮。
  • To override the shader used for Flares, open the Graphics window and set Light Halo to the shader that you would like to use as the override.
