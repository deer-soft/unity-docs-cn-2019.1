SceneManager.GetSceneByName

Switch to Manual
public static SceneManagement.Scene GetSceneByName (string name);

Parameters

name要查找场景的名称。

Returns

Scene 如果有效，则返回对场景的引用；否则返回无效场景。

Description

搜索已加载的场景，查找包含给定名称的场景。

名称不得包含 .unity 扩展名。名称可以是 Build Settings 窗口中所示名称的最后一部分，在这种情况下，将返回第一个匹配的场景。名称也可以是是 Build Settings 中显示的路径（仍然不得包含 .unity 扩展名），在这种情况下，将只返回精确匹配的场景。名称不区分大小写。
