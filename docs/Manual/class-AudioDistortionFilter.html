音频失真滤波器 (Audio Distortion Filter)

Audio Distortion Filter 可将来自音频源的声音或者到达音频监听器的声音进行失真处理。

属性

属性： 功能：
Distortion 失真值。范围从 0.0 到 1.0。默认值为 0.5。

详细信息

应用 Audio Distortion Filter 可模拟低质量无线电传输的声音。
