GUIElement.HitTest

Switch to Manual
public bool HitTest (Vector3 screenPosition, Camera camera= null);

Description

屏幕上的某个点是否在该元素内？

如果该 GUIElement 包含此 screenPosition，则返回 true。screenPosition 以屏幕坐标指定， 与 Input.mousePosition 属性返回的值类似。 如果未给定 /camera/，则假定为填充整个游戏窗口的摄像机。

注意，如果位置在该元素内部， 则即使该游戏对象属于忽略射线投射 (Ignore Raycast) 层，也将返回 true/ （通常，鼠标事件不会发送给 Ignore Raycast 对象）。

另请参阅：GUILayer.HitTest。
