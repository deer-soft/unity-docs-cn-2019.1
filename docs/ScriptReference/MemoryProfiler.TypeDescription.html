TypeDescription

struct in UnityEditor.MemoryProfiler

Switch to Manual

Description

托管类型的描述。

Variables

arrayRank如果类型为 arrayType，则将返回数组的等级。（对于一维数组为 1，对于二维数组为 2，等等）
assembly此类型加载自的程序集的名称。
baseOrElementTypeIndex此类型的基本类型，由编入 PackedMemorySnapshot.typeDescriptions 的索引指向该类型。
fields包含此类型所有字段的描述的数组。
isArray此类型是否为数组？
isValueType此类型是否为值类型？（如果并非值类型，则为引用类型）
name此类型的名称。
size此类型实例的大小（以字节为单位）。如果此类型为 arraytype，则描述数组中的单个元素将占用的字节数。
staticFieldBytes在创建快照时，存储此类型静态字段的字节的实际内容。
typeIndex此类型的 typeIndex。此索引是一个编入 PackedMemorySnapshot.typeDescriptions 数组的索引。
typeInfoAddress包含虚拟机内此类型描述的内存中地址。该属性可用于将堆中的托管对象与其对应的 TypeDescription 相匹配，因为托管对象的第一个指针指向其类型描述。
