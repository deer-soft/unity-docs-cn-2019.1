PlayerSettings

class in UnityEditor

/

Inherits from:Object

Switch to Manual

Description

Player Settings 是您为即将在 Unity 中构建的最终游戏定义各种参数的地方。其中一些值将用于您打开独立平台游戏时所启动的分辨率对话框。
