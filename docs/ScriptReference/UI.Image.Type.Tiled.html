Image.Type.Tiled

Switch to Manual

Description

显示切开的 Sprite，并将其可调大小的部分平铺而非拉伸。

平铺图像的行为方式与 Sliced 图像相似，区别在于将图像的可调大小部分重复而非拉伸。对于拉伸时外观不理想的精细 UI 图形，此功能很有用。

它使用 Sprite.border 的值确定每个部分（边界和中心）的平铺方式。

图像部分将在 Sprite 中重复对应的部分，直至整个部分被填充。边角部分不受影响，绘制方式与切片图像相同。各边将在其长度范围内重复。中心部分将沿着图像的整个中心重复。



图像部分将在 Sprite 中重复对应的部分，直至整个部分被填充。

请注意，如果您要平铺一个带边界的 Sprite 或打包的 Sprite，将生成一个用于创建瓦片的网格。网格的大小不得超过 16250 个四边形；如果您的平铺的瓦片数超过这个限制，瓦片将被放大，以确保生成的四边形数低于此限制。

为获得最佳效率，使用的 Sprite 应无边界且无打包，并确保将 Sprite.texture 包裹模式设置为 TextureWrapMode.Repeat。这些设置将阻止生成其他几何体。如果无法实现，请限制图像中的瓦片数。 也可以在 Sprite Editor 中设置切片。
