Toast.Create

Switch to Manual
public static WSA.Toast Create (string xml);
public static WSA.Toast Create (string image, string text);

Parameters

xml带有瓦片数据的 XML 文档。
image将显示在 Toast 通知中的图像的 URL；可以为空，在此情况下，将显示纯文本通知。
text要显示在 Toast 通知中的文本。

Returns

Toast 用于与已创建通知进一步配合使用的 Toast 对象，或者 null（如果 Toast 通知创建失败）。

Description

创建 Toast 通知。

Toast 通知是通过向 XML 文档提供其数据来创建。 第二个变体是一种便捷方法，可以显示带有文本的简单 Toast 通知，并且可以选择在其中显示图像。
