UnityWebRequest.GetTexture

Switch to Manual
Obsolete public static Networking.UnityWebRequest GetTexture (string uri);
Obsolete public static Networking.UnityWebRequest GetTexture (string uri, bool nonReadable);

Parameters

uri要下载的图像的 URI。
nonReadable如果为 true，脚本将无法访问纹理的原始数据。这可以节省内存。默认值：false。

Returns

UnityWebRequest 经适当配置以下载图像并将其转换为 Texture 的 UnityWebRequest。

Description

创建用于通过 HTTP GET 来下载图像的 UnityWebRequest，并基于检索到的数据创建 Texture。

Obsolete - instead use UnityWebRequestTexture.GetTexture.

此方法可创建一个 UnityWebRequest 并将目标 URL 设置为字符串 uri 参数。此方法不设置其他标志或自定义标头。

此方法将一个 DownloadHandlerTexture 对象附加到 UnityWebRequest。DownloadHandlerTexture 是一个经过优化的专用 DownloadHandler，用于存储要在 Unity 引擎中用作纹理的图像。与下载原始字节并在脚本中手动创建纹理相比，使用此类可显著降低内存重新分配。此外，纹理转换将在工作线程上执行。

此方法不会附加 UploadHandler 到 UnityWebRequest。

UnityWebRequest.GetTexture 已弃用。改用 UnityWebRequestTexture.GetTexture。
