如何创建聚光灯剪影？

Unity 在标准资源中附带了一些__光照剪影。将标准资源导入项目后，即可在 Standard Assets > Light Cookies__ 中找到这些光照剪影。本页面介绍如何自行创建剪影。

为场景添加大量视觉细节的好方法是使用剪影，这是用于控制游戏内光照精确外观的灰度纹理。剪影非常适合制作移动的云团以及给人一种茂密树叶的印象。光源组件参考页面上提供关于所有这些设置的更多信息，但重点是，为了使纹理可用于剪影，需要设置以下属性：

要为聚光灯创建光照剪影，请执行以下操作：

1.在 Photoshop 中绘制剪影纹理。图像应为灰度图像。白色像素意味着完全光照强度，黑色像素意味着没有光照。纹理的边界必须是完全黑色的，否则光会泄漏到聚光灯外。 1.在 Texture Inspector 中，将 Texture Type 更改为 Cookie 1.启用 __Alpha From Grayscale__（通过这种方式，您可以创建灰度剪影，Unity 会自动将其转换为 Alpha 贴图）
